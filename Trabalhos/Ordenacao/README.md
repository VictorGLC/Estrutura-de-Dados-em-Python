# Terceiro Trabalho de Implementação

## Exercício 1

Inicialmente você deve implementar em Python os algoritmos de ordenação:
- Ordenação por seleção
- Ordenação por inserção
- Heapsort
- Mergesort

Em seguida, utilize funções prontas do Python para gerar sequências numéricas aleatórias de tamanhos variados:
- 100 elementos
- 1.000 elementos
- 10.000 elementos
- 1.000.000 elementos

Utilize uma função que retorne o tempo de execução das funções e monte uma tabela com os resultados obtidos para cada algoritmo e para cada uma das sequências geradas.

Por fim, faça uma análise desta tabela considerando o tempo gasto e o tamanho das sequências.

## Exercício 2

Faça uma implementação de uma tabela de dispersão (Hashing) usando endereçamento aberto para resolução das colisões.
